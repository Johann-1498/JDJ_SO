# JDJ_SO
Simulador educativo de un sistema operativo simplificado.

## Requisitos del Sistema
- Java 8 o superior
- 2GB RAM mínimo

## Instrucciones de Uso
1. Ejecutar `compilar.bat` (Windows) o `./compilar.sh` (Linux/Mac)
2. Seguir las instrucciones en consola
3. Los resultados se guardan en `outputs/`

## Estructura del Proyecto
- `inputs/` - Archivos de procesos
- `outputs/` - Resultados y logs
- `src/` - Código fuente
